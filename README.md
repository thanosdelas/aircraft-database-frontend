## Aircraft Database Frontend App

To set up the backend visit https://github.com/thanosdelas/aircraft-database-rails-backend

### Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
